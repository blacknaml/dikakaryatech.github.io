SIM Diklat Rumah Sakit SIDIK

SIDIK adalah aplikasi mengintegrasikan proses perencanaan, penyelenggaraan, pengevaluasian dan pendokumentasian kegiatan Pendidikan dan Pelatihan berbasis teknologi informasi dan komunikasi.

Aplikasi ini untuk membantu panitia pelaksana diklat untuk mempermudah proses pendataan peserta diklat khususnya saat proses registrasi peserta, serta dapat memudahkan proses pelaporan diklat.

SIDIK merupakan alat bantu untuk mengelola data kepegawaian menjadi bahan informasi. Dimana setiap pegawai pernah ikut diklat agar dapat terdata di aplikasi tersebut sehingga jangan sampai yang mengikuti Diklat hanya orang atau pegawai itu-itu saja. Tujuan dari SIDIK sebagai alat bantu untuk mempersiapkan peserta diklat dan data alumni.

  • Berbasis Web

    Cukup menggunakan web browser di komputer atau smart phone.

  • Landing Page

    Bisa digunakan untuk halaman publik untuk informasi umum departemen Diklat.

  • Kategori Kordik, KEP, dan Diklit

    Manajemen Pelatihan In / Out House internal Karyawan Perusahaan (Diklit), Manajemen PKL, Magang, dan Data awal dari Instansi luar Perusahaan (Kordik), Manajemen Penelitian dari Instansi luar Perusahaan (KEP)

Sistem Informasi Manajemen Proyek SIMAPRO

SIMAPRO adalah software aplikasi berbasis web berfungsi untuk mencatat proyek, barang, customer dan suplier. Sehingga proyek, inventori barang dan data customer bisa dipantau secara mudah melalui komputer atau smartphone

SIMAPRO menggunakan sistem approval bertingkat, dan memiliki beberapa kelompok user yaitu Admin, Direksi, Purchasing, Accounting dan PimPro.

  • Berbasis Web

    Cukup menggunakan web browser di komputer atau smart phone.

  • Landing Page

    Bisa digunakan untuk halaman publik untuk informasi umum departemen Diklat.

  • Analisa PIECES

    metode analisis yang terdiri dari 6 indikator penilaian yaitu Performance, Information, Economy, Control, Eficiency, dan Service. Analisis ini merupakan cara untuk mengidentifikasi dan memecahkan permasalahan yang terjadi pada suatu sistem berjalan.
