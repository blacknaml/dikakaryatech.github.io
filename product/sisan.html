Satu Sistem Untuk Semua Kebutuhan

SISAN dirancang untuk menjembatani koordinasi tiga pilar utama dalam kegiatan praktek kerja lapangan agar berjalan sinergis.

Peserta Didik

Akses praktis bagi Ketua & Anggota kelompok untuk melengkapi administrasi. Memudahkan pengunggahan berkas syarat masuk serta pemilihan jadwal praktek secara mandiri.

✓ Unggah Dokumen ID Card & Syarat
✓ Pilih Jadwal & Ruang Kerja
✓ Unduh Sertifikat Kelulusan

Pembimbing (DM & Reguler)

Ruang kontrol bagi dokter/pembimbing untuk memantau perkembangan peserta. Melakukan verifikasi berkas secara digital dan mendiskusikan revisi secara *real-time*.

✓ Pemantauan Log Book Aktif
✓ Sistem Verifikasi Sekali-Klik
✓ Chat Asisten Berkas Terintegrasi

Administrator

Otoritas penuh bagi Admin Pembimbing & Admin Sistem dalam memegang kendali ekosistem. Mengatur kriteria instansi pendidikan, kuota departemen, dan memvalidasi kelulusan akhir.

✓ Kelola Master Data Instansi & Kampus
✓ Set Kriteria Kehadiran & Kelulusan
✓ Export Laporan Kegiatan Format Excel/PDF

Fitur Utama Aplikasi

Dukungan Teknologi Terbaik untuk Administrasi Praktek

Pemberkasan dan koordinasi jadwal praktek kerja dikelola dengan alur terpadu guna menjamin efektivitas kegiatan di lapangan.

View In-Browser

Lihat berkas secara langsung di *browser* tanpa perlu membuang waktu mengunduhnya. Mendukung penampil dokumen PDF yang responsif dan jernih.

Approval Berjenjang

Sistem verifikasi otomatis dan berjenjang. Penyetujuan dimulai dari pembimbing lapangan, koordinator diklat, hingga keputusan admin sistem secara terstruktur.

Jadwal & Ruang Presisi

Peserta dapat membuat jadwal praktek kerja sekaligus memilih ruang praktek secara langsung melalui kalender interaktif untuk menghindari bentrokan kuota.

Ruang Diskusi Interaktif

Terdapat fitur *chat* langsung antara pembimbing dan peserta didik pada setiap berkas. Klarifikasi atau instruksi revisi terdokumentasi dengan baik.

Cetak Sertifikat Otomatis

Sertifikat kelulusan praktek dalam format PDF akan diterbitkan oleh sistem secara otomatis begitu semua prasyarat berkas dan penilaian telah lengkap.

Integrasi RESTful API

Terhubung secara aman dan mudah dengan sistem universitas atau portal instansi mitra luar melalui antarmuka REST API modern yang terdokumentasi.
